Usage

  • Classes archives and style archives must be placed in a local folder following the Tex Directory Structure (TDS) as explained here.
  • In order to compile properly a UFSCthesis in TeXstudio, it is necessary to have (and update mikTeX) with the following archives (modified locally):
    • abnt.cls
    • BUUFSCthesis.sty0
    • ufsc-alf.bst
    • ufscThesis.cls

nomenclature package

  • include at the end of each paragraph lines like

\nomenclature[lk]{$K$}{Watson UOPK factor.}
\nomenclature[lt]{$T$}{Temperature, °C.}
\nomenclature[ls]{$SG$}{Specific gravity.}
\nomenclature[sb]{b}{Boiling point.}
\nomenclature[sa]{avg.}{Average.}
\nomenclature[a]{TBP}{True Boiling Point.}

  • The syntax is \nomenclature[xy]{symbol}{meaning}, where x is the type of abbreviation: a for acronyms, l for Latin letters, g for greek letters, s for subscripts, u for superscripts, z for others. y is a character used for organization of the list in alphabetical order.
  • + symbol, separates the acronyms from the other subgroups in the index.
  • For subgroups, the following code must be placed in the document header: 

\renewcommand{\nomgroup}[1]{%
	\ifthenelse{\equal{#1}{A}}{\item[\textbf{Acronyms}]}
	{\ifthenelse{\equal{#1}{G}}{\item[\textbf{Greek Letters}]}{}}
	{\ifthenelse{\equal{#1}{S}}{\item[\textbf{Subscripts}]}{}}
	{\ifthenelse{\equal{#1}{U}}{\item[\textbf{Superscripts}]}{}}
	{\ifthenelse{\equal{#1}{L}}{\item[\textbf{Latin Letters}]}{}}
	{\ifthenelse{\equal{#1}{Z}}{\item[\textbf{Others}]}{}}

  • Once the .tex file is compiled, a .nlo file is generated in the same root folder. then you must run the following command in the cmd window, in order to generate a .nls file.
makeindex ﬁlename.nlo -s nomencl.ist -o ﬁlename.nls
  • Then you must compile again, in order to print the lists.
  • Pages must be numbered.

brackets

  • unpaired \left( and \right) on any single line is not compiled. You can get round this with a \left. or a \right. at the beginning or end of each line where there is an unpaired bracket.

Macros

  • The sintaxis of new commands has the following estructure: \newcommand{name}[num]{definition}, for example

\newcommand{\wbalsup}[1] { This is the Wikibook about LaTeX supported by #1} 
\newcommand{\wbalTwo}[2] { This is the Wikibook about LaTeX supported by #1 and #2} 

% in the document body: 
\begin{itemize} 
    \item \wbalsup{Wikimedia} 
    \item \wbalsup{lots of users!} 
    \item \wbalTwo{John Doe}{Anthea} 
\end{itemize}

The output will be 

This is the Wikibook about LaTeX supported by Wikimedia 
This is the Wikibook about LaTeX supported by lots of users! 
This is the Wikibook about LaTeX supported by John Doe and Anthea

  • Use \newcommand* (permits the iclusion of commands in the arguments) and the command \def to get the user inputs: 

\newcommand*{\ResCountry}[1]{\def\@Rescountry{#1}}         %Defines the Residence country

Document differences via Latexdiff script

http://techshangrila.blogspot.com/2013/10/installing-latexdiff-of-windows.html
Prerequisite: Miktex (http://miktex.org/2.9/setup)
Prerequisite: Perl (can use Strawberry Perl: http://strawberryperl.com/releases.html)
Assume Miktex and Perl Interpreter are installed, and Miktex is installed in "C:\Program Files (x86)\MiKTeX 2.9\" (Other directory is fine, just adapt accordingly)
Go to "C:\Program Files (x86)\MiKTeX 2.9\scripts\latexdiff\"
Create a a new folder "perl" in "C:\Program Files (x86)\MiKTeX 2.9\scripts\latexdiff\"
Copy all files in "C:\Program Files (x86)\MiKTeX 2.9\scripts\latexdiff\" to new folder "C:\Program Files (x86)\MiKTeX 2.9\scripts\latexdiff\perl"
Rename all files (without extension) in "C:\Program Files (x86)\MiKTeX 2.9\scripts\latexdiff\perl", with an extension .ple.g., latexdiff to latexdiff.pl
Add "C:\Program Files (x86)\MiKTeX 2.9\scripts\latexdiff\perl" to the PATH environment (if you are not sure how to do so, see [here](http://www.java.com/en/download/help/path.xml))
Open Command Prompt, and type in
latexdiff.pl --flatten PreviousFile CurrentFile> diffe.g., latexdiff.pl --flatten "C:\old.tex" "C:\new.tex" > "C:\Diff.tex"
