EES

Created Saturday 18 August 2018

Features

  • It does not calculate the Liquid-vapour equilibrium.
  • The PFD must be generated by other program.
  • Properties of refrigerants and dioxanes available.
  • Can graph envelopes easily.
  • The lack of difference between phases makes it impossible to calculate the mixing exergy as the difference between the properties of the whole stream and the contribution of each component (in each phase) the use of can be inappropriate.
  • EES processes the statements in functions or procedures in the order they appear.
  • Upper and lower case letters are not distinguished.

Functions

  • A function can be used for picking only one output from a procedure. The function name must match the output name. for example: 

function H(C$,T)
{T: Temperature [K]} 
CALL JANAF(C$,T:CP,H,S) 
end

Functions must have arguments: you can use a joker variable in the function definition and then, use any value within the main code (when you dont need an output).

Modules/Subprograms

  • Modules/ Subprograms statement can (optionally) use a colon in the argument list to separate inputs from outputs. The number of arguments provided to the left of the colon is the number of degrees of freedom in the Module / Subprogram.
